## 📖 About

**MatMovies** is an upcoming free entertainment platform built to deliver a smooth, fast, and modern streaming experience.

With MatMovies, users will be able to enjoy thousands of entertainment contents from a single application.

### Available Content

- 🎬 Latest Movies
- 🇧🇩 Bangla Movies
- 📺 Bangla Natok
- 🌐 Web Series
- 🇺🇸 Hollywood Movies
- 🇮🇳 Bollywood Movies
- 🇰🇷 Korean Drama
- 🇯🇵 Anime
- 📡 Live TV *(Coming Soon)*
